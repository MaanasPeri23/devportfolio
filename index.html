Projects

Detecting TeraHertz Images Using Chanvese Algorithm

Terahertz Wavelength is non ionizing and can penetrate through human body and can detect metal objects. I Used Terahertz images produced from Terahertz cameras and gave them as an input to Chanvese Algorithm which in turn drew contour lines on top of it. Those images when given to Tensor Flow Algorithm using K Means Clustering gave the probability if the object was a threat object or not. I worked on this project for around an year in my Jr School year and applied to Central Sound Regional Engineering Fair and Washington State Science Engineering Fair.

Facial Recognition Using Open CV3 and Python3.

My project at Washington State Science Engeenring Fair using Tensor Flow gave an opening for me to work on OpenCv3 and Python3. This project was designed for GozoCabs as my Summer Project. Gozo Cabs is an Uber between Cities in India and serves 1000 plus cities. This Application helps them facial Id the drivers when the drivers change in between the long road trips and is a security feature which prevents unauthorized drivers from getting into the system.
